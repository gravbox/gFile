To run this installer open the project setting and click the "Debug" tab. 
Then in the "Start external program" box specifiy the .NET InstallUtil.exe

C:\Windows\Microsoft.NET\Framework\v4.0.30319\InstallUtil.exe

In the command line arguments box enter the name of this library.

Gravitybox.gFileSystem.Install.dll

You may then right click this project and select the menu "Debug|Start New Instance".
A dialog will appear that allows you to create a new database.
